확장기능(Extensions)

확장기능이란?

확장기능은 커맨드헬퍼의 펑션 이나 이벤트 를 추가해 주는 파일입니다.

마인크래프트 플러그인과 같이 .jar 확장자의 파일이며, (플러그인 폴더)/CommandHelper/extensions 에 넣으면 사용할 수 있습니다.

확장기능의 적용은 무조건 서버를 재시작 하여야 하며, 커맨드헬퍼의 자체 리로드 기능으로는 로드할 수 없습니다.

확장기능의 사용

확장기능 인덱스 에서 원하는 확장기능을 다운로드 받아,

extensions 폴더에 넣고 서버를 재시작합니다. 그리고 커맨드헬퍼의 스크립트 상에서 확장기능의 문서를 참고하여 확장기능의 펑션이나 이벤트를 사용할 수 있습니다.
