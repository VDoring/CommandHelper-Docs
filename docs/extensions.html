확장기능

확장기능은 CommandHelper 의 기능을 확장시켜 다른 플러그인 혹은 새로운 기능을 사용할 수 있도록 해 줍니다.

만약 확장기능이 이 목록에 없거나 링크가 깨지더라도 이곳 에서 혹은 검색으로 찾아보실 수 있습니다.

확장기능 이름 설명 문서(번역된 문서) 다운로드
CH-mcMMO mcMMO와 커맨드헬퍼를 연동시켜 주는 확장기능입니다. 문서(영어) 다운로드
CHCitizens Citizens과 커맨드헬퍼를 연동시켜 주는 확장기능입니다. 문서(영어) 다운로드
CHCombust 엔디티가 불타는(combust) 이벤트를 추가시켜 줍니다. 문서(영어) 다운로드
CHDangerous 불안정한 기능을 추가해 줍니다. 파일 쓰기가 필요할 때에는 CHFiles 사용을 추천합니다. 문서(영어) 다운로드
CHDisguises LibsDisguises API 와 커맨드헬퍼를 연동하여 엔디티를 변장시킵니다. 문서(영어) 다운로드
CHDynmap Dynmap와 커맨드헬퍼를 연동시켜주는 확장기능입니다. 문서(영어) 다운로드
CHFiles 커맨드헬퍼에 파일 관리 기능을 추가해 줍니다. 문서(영어) 다운로드
CHGCC 커맨드헬퍼에서 C/C++ 을 사용할 수 있습니다. 문서 없음 다운로드
CHGriefPrevention GriefPrevention API 와 커맨드헬퍼를 연동해 줍니다. 문서(영어) 다운로드
CHGroovy 커맨드헬퍼에서 Groovy 스크립트를 사용할 수 있습니다. 문서(영어) 다운로드
CHHTTPd 커맨드헬퍼에서 HTTP 서버를 실행합니다. 문서(영어) 다운로드
CHIRC 커맨드헬퍼에서 IRC 서버와 연결합니다. 문서 (영어) 예제(영어) 다운로드
CHJavascript 커맨드헬퍼에서 Javascript 를 사용할 수 있습니다. 문서 (영어) 다운로드
CHLogblock Logblock API 와 커맨드헬퍼를 연동해 줍니다. 문서 (영어) 다운로드
CHNaughty NMS 를 사용하는 펑션들을 추가해 줍니다. NMS 를 사용하므로 버전 영향을 많이 받습니다. 문서 (영어) 다운로드
CHNBT PowerNBT 와 연동하여 커맨드헬퍼에서 NBT 태그를 읽을 수 있도록 해 줍니다. 문서 (영어) 다운로드
CHPerms 커맨드헬퍼에서 권한 관리를 할 수 있도록 해 줍니다. 문서 (영어) 다운로드
CHPlaceholderAPI (PseudoKnight ver.) 플레이스홀더 API 에서 커맨드헬퍼 클로저를 실행할 수 있도록 합니다. 문서 (영어) 다운로드
CHPlaceholderAPI (TeamSteak ver.) 플레이스홀더 API 의 플레이스홀더를 커맨드헬퍼에서 사용하고 영구 저장소 네트워크의 값을 플레이스홀더에서 사용할 수 있도록 합니다. 문서 (영어) 다운로드
CHPlotMe PlotMe 와 커맨드헬퍼를 연동합니다. 문서 (영어) 다운로드
CHPlotSquared PlotSquared 와 커맨드헬퍼를 연동합니다. 문서 (영어) 다운로드
CHPluginManager 서버의 플러그인을 관리하는 펑션을 추가합니다. 문서 (영어) 다운로드
CHPlus ProtocolLib 을 통해 NMS 펑션을 커맨드헬퍼에서 사용합니다. 문서 (영어) 다운로드
CHProtocolSupport ProtocolSupport API 를 이용해 서버에 접속한 유저의 버전을 알아낼 수 있습니다. 문서 (영어) 다운로드
CHPython Jython 을 이용하여 커맨드헬퍼에서 Python 스크립트를 실행합니다. 문서 (영어) 다운로드
CHServerCommunication 커맨드헬퍼를 이용하여 서버간 통신을 구현합니다. 문서 (영어) 다운로드
CHSpigot 커맨드헬퍼에서 Spigot 전용의 기능을 사용할 수 있도록 합니다. 문서 (영어) 다운로드
CHStargate 커맨드헬퍼와 Stargate 플러그인을 연동합니다. 문서 (영어) 다운로드
CHTag 커맨드헬퍼와 iTag 플러그인을 연동합니다. 문서 (영어) 다운로드
CHVault 커맨드헬퍼와 Valut API 를 연동합니다. 문서 (영어) 다운로드
CHVirtualChests 커맨드헬퍼에서 가상 상자를 만들어서 플레이어에게 보여줄 수 있습니다. 문서 (영어) 다운로드
CHVotifierListener 커맨드헬퍼와 Votifier 를 연동합니다. 문서 (영어) 다운로드
CHWorldGuard 플레이어가 있는 월드가드 지역이 바뀌면 발생하는 이벤트를 추가합니다. 문서 (영어) 다운로드
SKCompat sk89q 의 플러그인과 연동하는 펑션들을 커맨드헬퍼에 추가해 줍니다. 문서 (영어) 다운로드
