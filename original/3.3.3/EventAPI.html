CmdlineEvents

Contains events related to cmdline events.

Event NameDescriptionPrefiltersEvent DataMutable Fields
cmdline_prompt_input Fired when a command is issued from the interactive prompt. If the event is not cancelled, the interpreter will handle it as normal. Otherwise, the event can be cancelled, and custom handling can be triggered.

command: The command that was triggered, shellMode

shutdown Fired the process is being shut down. This is not guaranteed to run, because some cases may cause the process to die unexpectedly. Code within the event handler should take as little time as possible, as the process may force an exit if the handler takes too long.
